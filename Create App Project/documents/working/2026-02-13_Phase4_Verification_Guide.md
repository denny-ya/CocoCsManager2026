# ✅ Phase 4: 검색/필터 및 UX 개선 검증 가이드

모든 기능 구현이 완료되었습니다. 아래 절차에 따라 앱이 정상적으로 동작하는지 확인해주세요.

## 1. 🔍 [검색] BS 및 리워크
- [ ] **필터 칩 확인**: 검색 결과 화면 상단에 `[전체] [완료] [진행중] [대기]` 칩이 보이는지 확인.
- [ ] **필터링 동작**: `[완료]` 칩을 눌렀을 때 완료된 항목만 남는지 확인.
- [ ] **새로고침**: 리스트를 아래로 당겼을 때(Pull to Refresh) 새로고침 아이콘이 돌면서 로딩되는지 확인.
- [ ] **스켈레톤 로딩**: 검색 버튼을 누른 직후 회색 박스(Skeleton)들이 깜빡이며 로딩 효과를 주는지 확인.

## 2. 🚛 배차 목록
- [ ] **날짜 선택**: 날짜 입력창을 터치하면 **하단 모달(Bottom Sheet)** 달력이 뜨는지 확인.
- [ ] **날짜 변경**: 달력에서 다른 날짜를 선택하고 `확인`을 누르면 입력창 날짜가 바뀌는지 확인.
- [ ] **조회 동작**: `조회` 버튼을 누르면 스켈레톤 로딩 후 결과가 표시되는지 확인. (Mock 데이터라 결과는 항상 같음)

## 3. 📍 영업점 주소 검색
- [ ] **스켈레톤 로딩**: 화면 진입 시 처음에 스켈레톤 로딩이 1초간 보이는지 확인.
- [ ] **FAB 버튼**: 리스트를 아래로 스크롤했을 때 우측 하단에 **위로 가기 화살표 버튼(FAB)**이 나타나는지 확인.
- [ ] **FAB 동작**: 버튼을 누르면 리스트 맨 위로 부드럽게 스크롤되는지 확인.

## 4. 📊 실적 및 통계
- [ ] **화면 이동**: `AS 실적` 메뉴를 눌러 그래프가 있는 화면으로 이동하는지 확인.
- [ ] **화면 이동**: `CS 통계` 메뉴를 눌러 Top 5 차트가 있는 화면으로 이동하는지 확인.
