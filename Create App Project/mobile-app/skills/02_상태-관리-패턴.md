## 1. usePersistedState — 상태 영속화

AsyncStorage를 사용해 앱을 닫아도 상태를 유지합니다.

```typescript
import { useState, useEffect } from 'react';
import AsyncStorage from '@react-native-async-storage/async-storage';

export function usePersistedState<T>(key: string, defaultValue: T) {
  const [state, setState] = useState<T>(defaultValue);
  const [loaded, setLoaded] = useState(false);

  // 저장된 값 불러오기
  useEffect(() => {
    AsyncStorage.getItem(key).then(stored => {
      if (stored !== null) setState(JSON.parse(stored));
      setLoaded(true);
    });
  }, [key]);

  // 값 변경 시 저장
  const setPersistedState = (value: T | ((prev: T) => T)) => {
    setState(prev => {
      const next = value instanceof Function ? value(prev) : value;
      AsyncStorage.setItem(key, JSON.stringify(next));
      return next;
    });
  };

  return [state, setPersistedState, loaded] as const;
}
```

### 사용 예시
```tsx
function SearchScreen() {
  const [recentKeywords, setRecentKeywords, loaded] = usePersistedState<string[]>(
    'recent_search',
    []
  );

  const handleSearch = (keyword: string) => {
    setRecentKeywords(prev => [keyword, ...prev.filter(k => k !== keyword)].slice(0, 10));
  };
}
```

### 적용 시점
- 최근 검색 기록 저장
- 사용자 설정/필터 값 유지
- 마지막 선택한 서비스 기억

> **설치 필요**: `npx expo install @react-native-async-storage/async-storage`

## 2. debounce — 디바운스

빠르게 연속 호출되는 함수의 실행을 지연합니다.

```typescript
export function debounce<T extends (...args: any[]) => any>(
  fn: T,
  ms: number = 300
): (...args: Parameters<T>) => void {
  let timeoutId: ReturnType<typeof setTimeout>;
  return (...args: Parameters<T>) => {
    clearTimeout(timeoutId);
    timeoutId = setTimeout(() => fn(...args), ms);
  };
}
```

### 사용 예시
```tsx
function StoreSearchScreen() {
  const [query, setQuery] = useState('');
  const [results, setResults] = useState([]);

  // 입력할 때마다 API를 호출하지 않고, 300ms 멈추면 검색
  const debouncedSearch = useMemo(
    () => debounce(async (keyword: string) => {
      const res = await searchStores(keyword);
      setResults(res);
    }, 300),
    []
  );

  return (
    <TextInput
      value={query}
      onChangeText={(text) => {
        setQuery(text);
        debouncedSearch(text);
      }}
    />
  );
}
```

- 영업점 검색 시 실시간 필터링
- 텍스트 입력 기반 검색이 추가될 때
