## 1. 배열 그룹화 — groupBy

데이터를 특정 기준으로 그룹화합니다.

```typescript
export function groupBy<T>(arr: T[], key: keyof T): Record<string, T[]> {
  return arr.reduce((groups, item) => {
    const value = String(item[key]);
    (groups[value] = groups[value] || []).push(item);
    return groups;
  }, {} as Record<string, T[]>);
}
```

### 사용 예시
```typescript
const results = [
  { id: '1', status: '완료', storeName: '가락점' },
  { id: '2', status: '진행중', storeName: '강남점' },
  { id: '3', status: '완료', storeName: '잠실점' },
];

const grouped = groupBy(results, 'status');
// { '완료': [{...}, {...}], '진행중': [{...}] }
```

### 적용 시점
- BS 검색 결과를 상태별 그룹화
- 영업점을 지역별 그룹화
- 배차를 시간대별 그룹화

## 2. 배열 정렬 — sortBy

여러 기준으로 정렬합니다.

```typescript
export function sortBy<T>(arr: T[], key: keyof T, order: 'asc' | 'desc' = 'asc'): T[] {
  return [...arr].sort((a, b) => {
    const valA = a[key], valB = b[key];
    if (valA < valB) return order === 'asc' ? -1 : 1;
    if (valA > valB) return order === 'asc' ? 1 : -1;
    return 0;
  });
}
```

### 사용 예시
```typescript
const sorted = sortBy(results, 'date', 'desc'); // 최신순
```

## 3. 비동기 배열 처리 — asyncForEach

여러 API를 순차적으로 호출합니다.

```typescript
export async function asyncMap<T, R>(
  arr: T[],
  fn: (item: T, index: number) => Promise<R>
): Promise<R[]> {
  const results: R[] = [];
  for (let i = 0; i < arr.length; i++) {
    results.push(await fn(arr[i], i));
  }
  return results;
}

// 병렬 버전 (동시 호출)
export function asyncMapParallel<T, R>(
  arr: T[],
  fn: (item: T, index: number) => Promise<R>
): Promise<R[]> {
  return Promise.all(arr.map(fn));
}
```

### 사용 예시
```typescript
// 4개 서비스를 동시에 검색
const allResults = await asyncMapParallel(
  [0, 1, 2, 3],
  (serviceId) => searchBSRework(serviceId, keyword)
);
```

### 적용 시점
- 여러 서비스를 한 번에 검색할 때
- 복수 시트에서 데이터를 가져올 때

## 4. 데이터 필터 체인 — filterChain

여러 필터 조건을 깔끔하게 적용합니다.

```typescript
export function filterBy<T>(
  arr: T[],
  filters: Partial<Record<keyof T, any>>
): T[] {
  return arr.filter(item =>
    Object.entries(filters).every(([key, value]) => {
      if (value === undefined || value === null || value === '전체') return true;
      return item[key as keyof T] === value;
    })
  );
}
```

### 사용 예시
```typescript
const filtered = filterBy(results, {
  status: selectedStatus,   // '완료' | '진행중' | '대기' | '전체'
  storeName: selectedStore,  // 특정 영업점 또는 undefined
});
```

### 적용 시점
- 검색 결과 필터 칩 기능
- 배차 목록 상태별 필터링
