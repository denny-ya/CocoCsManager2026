# 📱 메인 화면 UI 구현 단계 체크리스트 (Phase 3) - v2

> 기존 Apps Script 앱의 **실제 화면 구조**를 기반으로 작성되었습니다.
> 구조(메뉴/기능)는 기존 앱을 따르되, 디자인은 모던하게 새로 구현합니다.
> **예상 소요 시간**: 2~3일

---

## � 화면 구조도

```
홈 (메인 메뉴)
├── [검색] BS 및 리워크
│   └── 서비스1~4 선택 박스 → 검색 결과
├── 실적 및 통계
│   ├── AS 실적
│   └── CS 통계
├── 영업점 주소 검색
│   └── 검색바 → 영업점 목록
├── 배차 목록
│   └── 날짜 선택 → 배차 리스트
└── 업무 가이드
    └── 가이드 콘텐츠
```

---

## 🎨 디자인 방향 (기존 앱과 다르게 적용)

> 기존: 파란 헤더 + 흰색 버튼 리스트 (단순)
> 변경: **Material Design 3 + 카드 기반 모던 UI**

### 🎨 컬러 팔레트 (로고 기반)

| 용도 | 컬러 | HEX | 적용 위치 |
|------|------|-----|----------|
| **Primary** | 🟦 진한 파랑 | `#0051A2` | 상단 바, 주요 버튼, 활성 아이콘 |

| **Text Primary** | ⬛ 다크 그레이 | `#212121` | 본문 텍스트 |

### 추천 디자인 스타일

| 요소 | 기존 앱 | 새 디자인 방향 (User 요청 반영) |
|------|---------|-------------------------------|
| **홈 메뉴** | 텍스트 버튼 리스트 | **2열 그리드 카드 레이아웃** (사진 참고) |

| **애니메이션** | 없음 | 터치 시 스케일 효과 (클릭감) |
| **다크모드** | 미지원 | 자동 다크모드 지원 |

## 1. 🏠 홈 화면 (메인 메뉴) 구현 (~1.5시간)

> 앱 실행 시 첫 화면. 5개 메뉴로 각 기능 화면 이동.

- [x] **홈 화면 레이아웃** (`app/(tabs)/index.tsx`)
  - 상단: 앱 타이틀 "Coco CS Manager" + 인사 메시지
  - 중앙: 5개 메뉴 카드 (아이콘 + 제목 + 간단 설명)
  - 각 카드 터치 시 해당 화면으로 이동
- [x] **메뉴 카드 컴포넌트** (`components/MenuCard.tsx`)
  - Paper `Card` 활용
  - 아이콘 + 제목 + 설명 + 화살표
  - 터치 리플 효과
- [x] **메뉴 데이터 정의**
  ```
  1. 🔍 [검색] BS 및 리워크 → /bs-search
  2. 📊 실적 및 통계       → /statistics
  3. 📍 영업점 주소록       → /store-directory
  4. 🚛 배차 목록           → /dispatch
  5. 📖 업무 가이드         → /work-guide
  ```
