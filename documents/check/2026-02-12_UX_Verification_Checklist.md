## 1. 🔍 [검색] BS 및 리워크
### 1.1 스켈레톤 로딩 (Skeleton Loading)

### 1.2 필터 칩 (Filter Chips)
- [ ] 검색 결과가 있을 때만 상단에 필터 칩(전체, 완료, 진행중, 대기)이 표시되는가?
- [ ] `[진행중]` 칩 선택 시, 상태가 '진행중'인 항목만 리스트에 남는가?
- [ ] `[전체]` 칩 선택 시, 다시 모든 항목이 표시되는가?
- [ ] 칩 선택 시 색상이 변경되어 선택된 상태임을 인지할 수 있는가?

### 1.3 당겨서 새로고침 (Pull to Refresh)
- [ ] 리스트를 맨 위에서 아래로 당겼을 때, 새로고침 인디케이터(Spinner)가 표시되는가?
- [ ] 1~2초 후 인디케이터가 사라지며 데이터가 재로딩되는가? (Mock 데이터 기준 동일 데이터 로드)
